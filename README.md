# TangoKado

A flashcard app for learning the most common words in any language.

## Features

- **14 Languages** — Spanish, French, Portuguese, German, Italian, Dutch, Croatian, Polish, Turkish, Russian, Japanese, Korean, Chinese, Arabic
- **1,000 words per language** — ranked by frequency of use, with translations and example sentences
- **Flashcard mode** — tap to flip, swipe to navigate, mark words as known or unknown
- **Typing mode** — type the translation to test active recall
- **Progress tracking** — separate stats for flashcards and typing, accuracy percentages per word
- **Study customization** — word range selection, reverse mode (English to foreign), shuffle toggle
- **Streak counter** — tracks consecutive days of study
- **Text-to-speech** — hear native pronunciation for every word
- **Dark mode** — full light and dark theme support
- **Custom decks** — create your own flashcard decks

## Requirements

- iOS 17.0+
- Xcode 16+

## Tech Stack

- SwiftUI
- SwiftData
- AVFoundation (text-to-speech)

## Privacy

TangoKado collects no user data. All progress is stored locally on device. No analytics, no ads, no tracking.
